Html, PHP 如何上傳檔案？

Share Comments

要在 Html 有上傳檔案的介面
必須利用到

這樣就會產生大家熟悉的’選擇檔案’按鈕
接著利用表單把檔案送出去
如果表單中有 file 類型的 input
enctype=”multipart/form-data” 是必須要加的

PHP處理檔案

我們可以用 $_FILES 陣列來取得上傳的檔案
上傳之後會存在一個暫存資料夾
如果要把檔案存在伺服器, 記得利用 move_uploaded_file 移到你要的資料夾

$_FILES[“upload”][“name”]：上傳檔案的原始名稱。
$_FILES[“upload”][“type”]：上傳的檔案類型。
$_FILES[“upload”][“size”]：上傳的檔案原始大小。
$_FILES[“upload”][“tmp_name”]：上傳檔案後的暫存資料夾位置。
$_FILES[“upload”][“error”]：如果檔案上傳有錯誤，可以顯示錯誤代碼。

利用 CSS 達到 div 併排

Share Comments
  • CSS

假設我們現在有3個區塊想要並列一起

我們可以在 Css 中利用 float 來達成併排效果

這樣就可以達到效果
但這時候你可能會發現, 父階層的 div 怎麼沒有高度？
原因是 float 造成元素沒有撐起 div 高度

有三種方法解決

  1. 對父階層設定高度

  2. 使用 css clear
    在結束標籤前加一個 div(注意不是直接作用在父階層哦)

  3. 對父階層使用 css overflow
