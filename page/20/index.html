C. Watering Flowers

題目

題解

給兩座噴水池座標
和n朵花的座標
調整兩座噴水池的灑水半徑 r1, r2
使所有的花都可以被澆到
求r1^2 + r2^2 最小值

作法

利用pair 將每朵花和兩座噴水池的距離存起來
排序之後
列舉每個固定當作 r1, 然後去求最大的 r2
相加之後求最小值

註: 多排序一個點為(0,0)，是因為可能只需要 r1, r2 其中一個就好, 另外一個為 0

AC code

D. Polyline

題目

題解

給三個點的座標
求最少可以組成幾個線段
線段之間不可相交

作法

有三種可能

  1. 直線
    只會有一個線段，輸出1
  2. 含一個轉角
    注意第三個點必須在 x1~x2 或 y1~y2 的範圍之”外”
    否則會形成兩個轉角
  3. 兩個轉角
    輸出3
