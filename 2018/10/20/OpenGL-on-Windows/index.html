在 Windows 安裝 OpenGL

Share Comments

Contents

  1. 1. 參考

作業需求要用到 OpenGL, 原本想在 Ubuntu 上裝, 不過範例都是 VS project
就先在 Windows 上裝吧！

首先下載 Visual Studio 2017
然後要裝 C++ 桌面開發

看別人說這樣 OpenGL 的函式庫已經裝好了
因為我裝之前也沒 check, 不確定是不是原本就有裝了
可以去 C:\Windows\System32 看有沒有 opengl32.dll

接著我們需要 glut 函式庫, 因為 OpenGL 只負責需渲染, 不能產生視窗
可以到 OpenGL 官網下載:
https://www.opengl.org/resources/libraries/glut/glutdlls37beta.zip

裡面包含:

將 glut.h 放到 C:\Program Files (x86)\Microsoft Visual Studio\2017\Community\VC\Tools\MSVC\14.15.26726\include
將 glut.lib, glut32.lib 放到 C:\Program Files (x86)\Microsoft Visual Studio\2017\Community\VC\Tools\MSVC\14.15.26726\lib\x64
將 glut.dll, glut32.dll 放到 C:\Windows\System32

我看到有人說 64 位元要放到 SysWoW64, 不過我實際查了之後, 這個資料夾其實是放相容 32 位元的資料
至於為什麼這樣可以參考什麼是SysWow64

接著我們開啟一個空專案
打開 工具 > Nuget 套件管理員 > 管理方案的 NuGet 套件
選瀏覽, 搜尋 nupengl
然後兩個都安裝

這個是幹嘛的呢？
我節錄別篇的話

簡單來說就是不用去設定函式庫路徑那些
接著我們開一個新的 cpp 檔

如果執行有一個黑色矩形跑出來就成功了！
