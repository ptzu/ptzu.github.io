Uva 11389 - The Bus Driver Problem

Share Comments

Contents

題目

input

多筆測資, 每筆測資開頭輸入 n, d, r (1 <= n <= 100)(1 <= d <= 10000)(1 <= r <= 5)
下一列輸入 n 個早上路徑
再下一列輸入 n 個下午路徑

題解

有 n 位司機, 每位司機會被分配到一個早上和下午路徑
若是一天的總路程超過 d, 則每一單位長度要付 r 元
求如何安排使加班費最少

作法

大的配小的
