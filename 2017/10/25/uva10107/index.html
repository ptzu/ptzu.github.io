Uva 10107 - What is the Median?

Share Comments

Contents

  1. 1. input
  2. 2. 題解
  3. 3. 作法
  4. 4. AC code

題目

input

0 <= X <= 2^31
N < 10000

題解

輸入一個數 X, 並將目前已輸入的數排序
根據每個輸入, 輸出當前的中位數

作法

若是每次讀進來一個都排序一次, 那肯定 TLE
所以我們可以利用 priority_queue, 每次進來只需要花 O(logn) 做調整
