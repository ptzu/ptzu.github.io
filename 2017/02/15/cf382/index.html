B. Urbanization

題目

input

人數： 1<= n, n1, n2 <= 10^6, n1 + n2 <= n
財富： 1 <= a <= 10^6

題解

現有 n 人, 想搬到兩座城市
這兩座城市分別可容納 n1, n2人
每個人都有財富 a
政府希望兩座城市分別的財富算術平均值最大
求兩座城市相加的算術平均值

作法

先把人數少的城市塞滿

C. Tennis Championship

題目

input

人數： 2 <= n <= 10^18

題解

有 n 個人參加網球賽, 採單淘汰制

比賽規定為對手已比賽場次, 不能比自己少超過一場

也就是兩場以上就不行

求要贏得冠軍, “最多”可以參加幾場比賽

作法

假如是問最少場次, 就是 log(n) 場(每次減少一半)

至於最多場次, 可以畫一下圖

當要贏 1 場時, 需要 2 人

2 場需要 3 人

3 場需要 5 人

可以發現其實就是 fibunacci 數列

原因是你要多贏一場

需要的人數是目前需要的, 以及一位勝場少你一場的對手

fib(n + 1) = fib(n) + fib(n - 1)

D. Taxes

# math
題目

input

年收入： 2 <= n <= 2 * 10^9

題解

某城市的稅收制度為年收入 n 的最大因數(不包含n)
可以將你的年收入拆成好幾份(分開收稅)
求最少需要繳多少稅

作法

根據 Goldbach’s conjecture(哥德巴赫猜想)

所以對於偶數而言, 先判斷是不是偶數
對於奇數而言：

  1. 是質數： 輸出 1
  2. n - 2是質數： 輸出 2(因為質數 n-2 是 1, 2 也是 1)
  3. 輸出 3(根據猜想)
