C. Hard Process

# binary_search # two points
題目

題解

給定長度為 n 只有 0, 1 的數列

現有 k 次機會可將 0 轉換成 1

求可連續的 1 最長是多少

作法

可利用二分搜, 假設長度 x 是否能在 k 次轉換得到

由於觀察到長度愈長, 愈難達成

有此單調性, 故可用二分搜

是 1 1 1 1 0 0 0 0 類型
