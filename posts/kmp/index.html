KMP Algorithm 探討

Jul 28, 2016
1 min read
String

關於KMP

一般再搜尋字串時, 暴力搜尋將每個位置都搜尋一次 同時比對要搜尋的字串(pattern) 時間複雜度為O(nm)

KMP演算法改進了一些不必要的搜尋 使得時間複雜度為O(n + m)

名詞定義

失敗函數(failure function): 當比對失敗時, pattern需位移幾位才能成為前綴, 一般用在字串比對失敗時查找, 故稱失敗函數

建立 F 函數

這邊放上網路上常見兩種寫法 不變的是 j 代表的是目前比對的字元的 ‘前一個字’ f[i] 代表比對失敗時, j 要變成的位置

  1. 字串從 0 開始存, 沒匹配字元時, f 值為 -1
void getF(string s){
    f[0] = -1;
    for(int i = 1; i < s.length(); i++){
        int j = f[i - 1];
        //如果不相等, 先位移到適合位置
        while(j >= 0 && s[j + 1] != s[i])
            j = f[j];

        if(s[j + 1] == s[i])
            f[i] = j + 1;
        //還是不相等, 此時 j = -1
        else
            f[i] = j;
    }
}
  1. 字串從 1 開始存, 沒匹配字元時, f 值為 0
